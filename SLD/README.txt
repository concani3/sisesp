Arquivos .ftl
-------------
São arquivos templates do Geoserver para criar respostas HTML customizadas de solicitações GetFeatureInfo.
No caso em questão, foi utilizado dentro da pasta GEOSERVER_DATA_DIR/workspaces/<workspace>/
Vide detalhes no manual do Geoserver em http://docs.geoserver.org/latest/en/user/tutorials/GetFeatureInfo/index.html


Arquivos .sld
-------------
São arquivos de "Styled Layer Descriptor" do Geoserver utilizados para especificar cores, traços e outros atributos usados para renderizar dados no mapa.
Vide detalhes no manual do Geoserver em http://docs.geoserver.org/latest/en/user/styling/sld-introduction.html
